# Token Refresh 双 Token 改造 — 完整实施步骤

## 1. 概念速览

当前（单 Token）:

-   登录 → accessToken(24h) + Redis会话(30min滑动) → 过期后必须重新登录

改造后（双 Token）:

-   登录 → accessToken(15min) + refreshToken(7天)
-   → accessToken 过期 → 前端用 refreshToken 静默换取新的 accessToken+refreshToken
-   → 用户无感，直到 refreshToken 也过期才需重新登录
-   → 登出时将 accessToken 的 jti 加入黑名单，立即失效

```mermaid
┌────────┐  accessToken(15min)  ┌────────┐
│  前端   │ ──────────────────> │  后端   │  正常业务请求
│        │ <────────────────── │        │  正常响应
│        │                     │        │
│        │  accessToken 过期    │        │
│        │ ──────────────────> │        │  401
│        │ <────────────────── │        │
│        │                     │        │
│        │  refreshToken       │        │
│        │ ──────────────────> │ /api/member/refreshToken
│        │ <────────────────── │  新 accessToken + 新 refreshToken
│        │                     │  (旧 refreshToken 立即作废 = 旋转)
│        │  重放原请求(新token)  │        │
│        │ ──────────────────> │        │  正常响应
└────────┘                     └────────┘
```

```java
/**
 * 从 Token 中提取 jti（JWT ID）。
 */
public String getJtiFromToken(String token) {
    try {
        return getClaimsFromToken(token).get(CLAIM_KEY_JTI, String.class);
    } catch (Exception e) {
        return null;
    }
}

/**
 * 从 Token 中提取过期时间。
 */
public Date getExpirationFromToken(String token) {
    try {
        return getClaimsFromToken(token).getExpiration();
    } catch (Exception e) {
        return null;
    }
}
```

#### 3.5 新增 `buildToken` 重载 — 支持自定义过期秒数

```java
private String buildToken(Map<String, Object> claims, String subject, long expirationSeconds) {
    return Jwts.builder()
            .setClaims(claims)
            .setSubject(subject)
            .setExpiration(new Date(System.currentTimeMillis() + expirationSeconds * 1000))
            .signWith(SignatureAlgorithm.HS512, secret)
            .compact();
}
```

并将原 `buildToken` 改为委托：

```java

    return buildToken(claims, subject, expiration);
}
```

#### 6.3 实现 `refreshToken` 方法

```java
@Override
public MemberLoginVO refreshToken(String refreshToken) {
    // 1. Redis 查找 refreshToken → phone
    String refreshKey = RedisKeyConstant.API_REFRESH_TOKEN_PREFIX + refreshToken;
    String phone = stringRedisTemplate.opsForValue().get(refreshKey);
    if (phone == null) {
        throw new IllegalArgumentException("登录已过期，请重新登录");
    }

    // 2. 查询会员状态
    UmsMember member = getByPhone(phone);
    if (member == null || member.getStatus() == null || member.getStatus() != 1) {
        stringRedisTemplate.delete(refreshKey);
        throw new IllegalArgumentException("账号异常，请重新登录");
    }

    // 3. 生成新 accessToken（短有效期）
    String newAccessToken = jwtTokenUtil.generateToken(phone, accessTokenExpiration);

    // 4. 生成新 refreshToken（旋转：旧的立即作废）
    String newRefreshToken = UUID.randomUUID().toString();
    stringRedisTemplate.delete(refreshKey);
    stringRedisTemplate.opsForValue().set(
            RedisKeyConstant.API_REFRESH_TOKEN_PREFIX + newRefreshToken,
            phone, refreshTokenExpiration, TimeUnit.SECONDS);

    // 5. 刷新 Redis 会话
    ApiMemberDetails memberDetails = new ApiMemberDetails(member);
    RedisUtil.set(RedisKeyConstant.API_MEMBER_INFO_KEY + phone,
            memberDetails, 30, TimeUnit.MINUTES);

    // 6. 组装返回
    MemberLoginVO vo = new MemberLoginVO();
    vo.setToken(newAccessToken);
    vo.setTokenHead(tokenHead);
    vo.setRefreshToken(newRefreshToken);
    vo.setMember(member);

    log.info("C 端用户 Token 刷新成功: phone={}", maskPhone(phone));
    return vo;
}
```

```java
/**
 * 登出：清除 Redis 会话 + accessToken 加入黑名单 + 删除 refreshToken。
 */
@PostMapping("/logout")
public Result<Void> logout(HttpServletRequest request,
                           @RequestParam(required = false) String refreshToken) {
    UmsMember member = getCurrentMember();
    // 1. 删除 Redis 会话
    RedisUtil.delete(RedisKeyConstant.API_MEMBER_INFO_KEY + member.getPhone());

    // 2. 当前 accessToken 的 jti 加入黑名单（TTL = JWT 剩余有效期）
    String authHeader = request.getHeader("Authorization");
    if (authHeader != null && authHeader.startsWith("Bearer ")) {
        String token = authHeader.substring(7);
        String jti = jwtTokenUtil.getJtiFromToken(token);
        Date expiration = jwtTokenUtil.getExpirationFromToken(token);
        if (jti != null && expiration != null) {
            long remainingSeconds = (expiration.getTime() - System.currentTimeMillis()) / 1000;
            if (remainingSeconds > 0) {
                RedisUtil.set(RedisKeyConstant.API_TOKEN_BLACKLIST_PREFIX + jti,
                        "1", remainingSeconds + 1, TimeUnit.SECONDS);
            }
        }
    }

    // 3. 删除 refreshToken
    if (refreshToken != null && !refreshToken.isEmpty()) {
        RedisUtil.delete(RedisKeyConstant.API_REFRESH_TOKEN_PREFIX + refreshToken);
    }

    return Result.success();
}
```
